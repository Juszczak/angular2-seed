# Angular seed

Seed project for Angular2 TypeScript Apps :rocket:


## Setup
Install [nvm](https://github.com/creationix/nvm):
```
brew install nvm
```

Install most recent `node` version using:
```
nvm install 6
```

Use version specified in `.nvmrc`:
```
nvm use
```

Install dependencies by typing:
```
npm i
```
